## 🚀 Running the project

### Back-end

Clone the project

```bash
  git clone https://github.com/skmonsterr/ignite-delivery
```

Enter the project directory

```bash
  cd ignite-delivery
```

Install with dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
