## 🚀 Running the project

### Back-end

Clone the project

```bash
  git clone https://link-para-o-projeto
```

Enter the project directory

```bash
  cd ignite-delivery
```

Install with dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
